## Code Snippets

### Generate Express site skeleton

```
npx express-generator --ejs bulletin_board
```
[Express Getting Started](https://expressjs.com/en/starter/generator.html)

### Run a script

```
node [file]
```

but usually

```
npm install
npm start
```

### Debug

Start node with a debugger:

```
node debug [file]
```

or;

```
node debug ./bin/www
```

Attach in Chrome DevTools:

```
about:inspect
```
